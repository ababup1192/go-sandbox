# Go Sandbox

## Requirement

- [direnv](https://github.com/direnv/direnv)
- [dep](https://github.com/golang/dep)
- [robo](https://github.com/tj/robo)

## Install

```
$ brew install direnv
$ go get -u github.com/golang/dep/...
$ go get github.com/tj/robo
```

## Usage

```
$ robo init
$ robo test
$ robo build
$ robo buildRun
```
